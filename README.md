# DevDash

> A sleek, developer project hub for tracking side projects, capturing dev notes, and organizing code snippets — all in one place.

---

## Overview

**DevDash** is a personal dashboard built for developers who juggle multiple side projects. Instead of scattering notes across Notion, code across GitHub gists, and ideas across sticky notes, DevDash consolidates everything into a single, beautifully crafted workspace.

Whether you're debugging a tricky bug, saving a useful code snippet, or tracking progress on your next SaaS idea, DevDash keeps your developer life organized.

---

## Features

### Project Tracking
- Create and manage projects
- Track status: **Planning**, **Active**, **Paused**, or **Completed**
- Tag projects with tech stack (React, TypeScript, Python, etc.)
- Link GitHub repositories and live demo URLs
- Dashboard stats overview at a glance

### Developer Notes
- Capture dev notes scoped to each project
- Full CRUD operations with inline editing

### Code Snippets Vault
- Save reusable code snippets with syntax highlighting
- One-click copy to clipboard
- Filter by language, search by keyword
- Optionally link snippets to specific projects

### Global Search
- Instant search across projects, notes, and snippets
